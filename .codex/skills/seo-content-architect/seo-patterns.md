# SEO Patterns

## Metadata Template

- Title: `<primary keyword>: clear benefit or page type`
- Meta description: 120-155 characters when possible; mention the use case and emotional value.
- H1: close to the primary keyword, written naturally.
- Canonical: exact public route for the page.
- Open Graph title: warmer and more shareable than the SEO title.
- Open Graph description: describe why someone would open the deck.

## Heading Pattern

- H1: Search intent.
- H2: Immediate question collection.
- H2: How to use these questions.
- H2: Related question decks.
- H2: Optional FAQ if visible and genuinely useful.

## Internal Link Pattern

Link by adjacent social context:

- General chat -> friends, icebreakers, deep.
- Couples -> first date, deep, travel.
- Travel -> roadtrip, friends, couples.
- Boredom -> random, friends, chat.
- Icebreakers -> first date, chat, friends.

Use descriptive anchor text, not "click here".
