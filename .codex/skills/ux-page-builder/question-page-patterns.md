# Conversational Question Page Patterns

## Recommended Structure

1. H1 that matches search intent.
2. Short human intro.
3. Immediate question card or deck selector.
4. Copy/share/random controls.
5. Indexable question list.
6. Related categories.
7. Optional visible FAQ only if useful.

## Category Tone Notes

- Couples: soft language, warm spacing, intimate but safe prompts.
- Friends: quicker rhythm, playful labels, casual browsing.
- Travel: movement-oriented copy, road/trip framing, discovery.
- Boredom: low commitment, random mode, playful surprises.
- Deep: calmer pacing, respectful framing, no pressure.
- Icebreaker: clear social safety, easy first questions.

## Question Card Checklist

- Large readable question text.
- Stable card dimensions on mobile.
- Obvious next question control.
- Copy/share controls with accessible labels.
- Category or intensity visible when useful.
- Works without hover.
